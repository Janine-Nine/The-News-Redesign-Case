# 📰 The News — Front-end Redesign

# THE-NEWS

O maior jornal imparcial digital do país

Redesign da plataforma **The News**, desenvolvido como projeto de estudo e desafio técnico para vagas de **Front-end Developer** e **UI/UX Designer**.

O objetivo do projeto é recriar a experiência do usuário com uma interface moderna, responsiva e organizada, utilizando boas práticas de desenvolvimento Front-end e Design de Interfaces.

---

# ✨ Objetivos

* Recriar a interface do The News
* Desenvolver uma aplicação responsiva
* Aplicar conceitos modernos de UI Design
* Organizar o projeto utilizando componentes reutilizáveis
* Preparar a aplicação para futura integração com APIs
